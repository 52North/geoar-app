# GeoAR App

An augmented reality client for live spatial data.

## Documentation

https://wiki.52north.org/bin/view/Projects/GeoAR

## Development

``mvn clean install``, optionally with ``-Dandroid.sdk.path=$ANDROID_HOME``

Maven version 3.1.1 is required.
